# J.A Fast Delivery Monorepo

This is the development monorepo for J.A Fast Delivery. A premium, gamified logistics platform.

## Structure

- **apps/web**: The main customer-facing website (HTML, CSS, JS).
- **packages/**: Shared UI libraries (future expansion).

## Features

- **Premium UI**: Dark mode with neon accents (`#b6ff27`).
- **Gamified Elements**: Interactive stats, hover effects, hexagon grid.
- **Performance**: Vanilla JS for maximum speed and zero build-time overhead.

## Development

To run the project locally:

1. Open `apps/web/index.html` in your browser.
   OR
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev` (requires `serve`)

## Design System

- Primary Font: Outfit (Google Fonts)
- Accent Color: `#b6ff27`
- Background: `#1f1f1f`
